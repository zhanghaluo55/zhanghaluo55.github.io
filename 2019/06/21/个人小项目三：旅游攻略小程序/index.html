个人小项目三：旅游攻略小程序

发表于 2019-06-21 | 分类于 实战项目 | | 本文总阅读数 次
字数统计 字; 747 | 阅读时长 分钟 2

本项目立足于粤港澳大湾区开发的一个旅游攻略软件，旨在利用小视频技术，VR图片展示等形成一种新型的具象化的游玩生活攻略,一定程度解决当前攻略软件偏向文字图片结合的表述形式单一不够吸引人的问题，结合新进的大数据分析技术为用户提供在游玩路线推荐和相应时间点的路况。针对日益严重的“低头族”现象，我们推出“成就系统”，“附近的玩伴”等功能鼓励用户的线下交流，通过这种虚拟和现实结合的方式提供用户丰富用户的现实生活，促进粤港澳大湾区的融合以及推广，强化粤港澳大湾区的影响力。

运用的技术

  • 前端
    jsp,js,css,微信小程序
  • 后端
    java语言，ssm框架
  • 开发环境
    windows系统,Android,mysql,myeclipse,navicat,微信开发者工具

#团队介绍

我们团队的有着较强的综合实力。首先团队成员都有着丰富的想象力，这为项目主题的 提出奠定了坚实的基础。另外团队组建的人员涉猎广泛，我们团队成员里面有擅长写代码的，擅长于美工的，有熟悉算法、网络、数据库的，有严谨细致的文档及视频制作人员。最重要的是我们团队的成员都有吃苦耐劳的恒心和毅力。项目开发项目涉及知识面广，工作量挺大。硬件上的限制给我们造成了巨大的挑战。面对没有资料，模块不齐全， 以及算法和技术方面无法支撑起项目的构思时，我们便在一起不断思考解决方案，争取通过好的软件的开发技术和想象力来弥补硬件和技术的不足。

总结

本项目前端采用微信小程序开发，后端采用ssm框架搭建，我主要负责个人中心和成就系统功能模块的开发。本项目旨在利用小视频技术，VR图片展示等形成一种新型的具象化的游玩生活攻略，结合新进的大数据分析技术为用户提供在游玩路线推荐和相应时间点的路况。通过本项目，丰富了微信小程序的知识，学会了协调团队开发，作为队长更多的巩固了开发文档的编写和分配队员任务，程序开发能力进一步提高。
