个人小项目一：汽车销售网站

发表于 2019-06-20 | 分类于 实战项目 | | 本文总阅读数 次
字数统计 字; 428 | 阅读时长 分钟 1

本系统为javaweb项目，采用hibernate,struts2框架，划分成前台和后台两部分。前台主要为用户提供车辆信息浏览、在线车辆订购并完成支付过程。后台则负责对用户、车辆、订单的信息进行管理。为不同用户类型分别提供了不同的功能，为管理员提供登陆，管理用户、商品的信息的功能；为销售人员提供登陆，负责管理商品，有分类商品、上架和修改商品，管理订单的功能。

运用的技术

  • 前端
    jsp，js，jquary，c标签
  • 后端
    java语言，struts2框架，hibernate框架
  • 开发环境
    windows系统,mysql,eclipse,navicat

总结

本项目由我和另一个小伙伴完成，我主要负责用户使用的前台界面前端和后端功能的实现。网站的功能基本完备，包括车辆信息的展示、查找、排序、购买，个人中心功能，订单功能。通过本项目，加深了对hibernate,struts2框架的理解，以及开发的一些技巧，补充了前端的知识。个人开发比较费时，比较考验耐心，但沉住气都会有所收获，都能积累自己的开发经验。
